HBO Clone

Streaming app clone built with NextJS. Application uses ContextAPI for global state. It utilizes the "themoviedb" api to display all the movie related content with help of axios. Many different endpoints of api were used to display different type of content and information. Search feature looks for movies/series based on the keyword typed in search bar. Movies can also be saved (and removed) in "My List" section which can be found in the right corner by clicking fake account name. This is accomplished by use of localstorage.
